# create-gvc-app

The CLI for **The Playground** — a builder toolkit for the Good Vibes Club community.

```sh
npx create-gvc-app
```

The CLI walks you through everything: name your project, pick what you want to build, describe your idea, choose extras (blockchain data, badges, leaderboards, more). Your project is ready with the full GVC brand system, a Claude guide, and everything you need to start building.

### Full docs, templates, and web builder

- Repo: https://github.com/brydisanto/gvc-builder-kit
- Web builder (no terminal needed): https://goodvibesclub.ai
- Prompt machine: https://goodvibesclub.ai/prompt-machine
